Detailed Description

A class that implements a semaphore.

This class implements a semaphore, which is a synchronization mechanism used to protect shared resources or to synchronize threads. It provides two basic operations, Wait and Signal.

enum Semaphore::LUCIDSemaphoreType

Enumerated type for possible semaphore types.

There are three types of Semaphore:

  • kNone (No type)
  • kIn (IN queue Semaphore)
  • kOut (OUT queue Semaphore)
Enumerator

Standard semaphore type if not assigned.

Semaphore for the IN queue inside the MemoryManager.
