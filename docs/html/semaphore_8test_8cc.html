Test:
Tests the Wait and Signal functions of the Semaphore object with multiple threads. Creates a Semaphore object with an initial count of 0 and starts 10 threads that wait on the semaphore and signal it. The main thread signals the semaphore 10 times to wake up the waiting threads, waits for the threads to complete, waits on the semaphore and signals it once. Then it tests if the count of the semaphore is 10.
