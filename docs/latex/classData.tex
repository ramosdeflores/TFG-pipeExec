\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classData}{Data}} class that provides access to various forms of manipulation and allocation for the processing units. 

This class provides functionallity to store multiple data via keys and a pointer to the original data that was stored initially with the creation of the class.
