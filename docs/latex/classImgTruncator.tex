\doxysubsection{Detailed Description}
This processing unit gets an array of heights and sets all values as multiples of 100. 

The class needs a width Data\+Key pair inside the \doxylink{classData}{Data} object and a height Data\+Key for it to know the dimensions of the image. It also divides all the numbers by 100 and multiplies them by the same number so the output is normalized to multiples of 100.
