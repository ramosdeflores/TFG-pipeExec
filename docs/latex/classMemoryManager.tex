\doxysubsection{Detailed Description}
A memory manager that provides an interface for managing a fixed-\/size queue of pointers to memory buffers. 

This class provides functionality to push and pop memory buffers in and out of the queue, as well as a debug flag to print the status of the \doxylink{classMemoryManager}{Memory\+Manager}. It also implements semaphores to ensure synchronization between the producer and consumer threads.
