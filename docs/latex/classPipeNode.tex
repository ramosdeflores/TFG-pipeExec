\doxysubsection{Detailed Description}
Class representing a single node in a processing pipeline. 

This class represents a single node in a processing pipeline. It contains a processing unit interface which is used to process the input data from the previous node, and produce output data for the next node. It also contains input and output memory managers for communication with the previous and next nodes. Each node is executed in its own thread.
