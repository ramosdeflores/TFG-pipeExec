\doxysubsection{Detailed Description}
Class representing the pipeline for the processing of any type of data. 

This class masks the processing of the data allowing the (main) thread to only see one queue of data while inside the \doxylink{classPipeline}{Pipeline} there are multiple queues and threads consuming the data and processing it concurrently to speed up the proccess.
